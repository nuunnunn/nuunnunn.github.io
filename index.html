One page

HTML CSS jQuery

팀버튼 감독님의 이야기와 필모그래피, 미술 작품을 담은 사이트입니다.

제가 생각한 작품들의 분위기에 맞게 각각의 섹션을 구성했습니다.

흑백의 이미지에서 Hover하면 원래의 이미지 색상이 나오고

사이트를 만들며 어릴적 추억도 상기하고 감독님에 대한 새로운 정보도 알게되어 재밌는 시간이었습니다.
